Il est possible d'inclure une image en utilisant du code Scilab (il faut déclarer au préalable le namespace xmlns:scilab="http://www.scilab.org"):

<scilab:image>

</scilab:image>

Si une image contient une chaîne de caractères localisées, il faut alors ajouter l'attribut scilab:localized="true":

<scilab:image localized="true">
  plot2d();

</scilab:image>
